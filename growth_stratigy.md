## 1. 현황 진단

### 1-1. 사이트 개요
- **서비스**: 소아 해열제(타이레놀, 부루펜, 챔프시럽, 맥시부펜 등) 체중별 용량 계산기
- **기술 스택**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **도메인**: kidsfever.xyz
- **언어**: 한국어 + 영어 (next-intl)
- **주요 페이지**: 계산기(홈), 블로그, FAQ, 소개, 연락처, 약관/개인정보처리방침

### 1-2. 트래픽 현황

#### 네이버 서치어드바이저 (1/4~2/2, 30일)
| 구분 | 일평균 노출 | 일평균 클릭 | CTR | 비중 |
|------|-----------|-----------|-----|------|
| 모바일 | 340 | 16 | 4.7% | 94% |
| PC | 21 | 1.2 | 5.7% | 6% |
| **합계** | **361** | **17** | **4.8%** | 100% |

- 월간 네이버 유입: **~500 클릭**
- Google 검색 유입: 별도 (미측정, 추가 유입 있을 것으로 추정)

#### KakaoAdFit 실측 데이터 (1/26~2/2)
| 지표 | 수치 |
|------|------|
| 일 광고 요청 | 7~43건 |
| Fill Rate | 0~94% |
| 일 광고 노출 | 0~31건 |
| VR (Viewable Rate) | 63~92% |
| **광고 클릭** | **0건 (전 기간)** |
| **수익** | **0원 (전 기간)** |

> 1/21~1/25 기간은 요청수 자체가 0 — 이 기간에 광고 스크립트가 작동하지 않았거나 사이트 접근이 없었음.

### 1-3. 수익화 채널 현황

| 채널 | 상태 | 상세 |
|------|------|------|
| **KakaoAdFit** | 승인 완료, 수익 0원 | CPC 모델이나 클릭 0건. 원인: 절대 트래픽 부족 |
| **Google AdSense** | 사이트 미승인 | `ca-pub-5648788643644962` 스크립트만 로드. 광고 유닛 코드 없음 |
| **Pieceful 배너** | 활성 | 자체 앱 크로스 프로모션 (직접 수익 아님) |

### 1-4. 수익 0원의 근본 원인

```
근본 원인: 절대적 트래픽 부족
  └─ 일 방문자 ~20~40명 수준
      └─ 일 광고 노출 ~15~30건
          └─ 디스플레이 광고 평균 CTR 0.5~2% 적용 시
              └─ 기대 일 클릭수: 0.08~0.6건 → 대부분의 날 0건이 정상
                  └─ CPC 모델에서 클릭 0 = 수익 0
```

KakaoAdFit 설정, 매체 심사, 계정 상태에는 문제 없음.
Fill Rate 56~98%로 광고 자체는 송출되고 있으며, VR도 63~92%로 광고 가시성도 양호.
**광고 수익을 만들려면 트래픽을 최소 10배 이상 늘려야 함.**

## 3. 성장 전략 로드맵

### Phase 1: 기반 정비 — 광고 인프라 구축

#### 1-A. Google AdSense 승인 획득
**현 상태**: 스크립트는 코드에 있으나 사이트 미승인.

**YMYL 사이트 AdSense 승인 체크리스트:**
| 요건 | 상태 |
|------|------|
| 충분한 콘텐츠 | ✅ 블로그 + 계산기 + FAQ |
| 개인정보처리방침 | ✅ |
| 이용약관 | ✅ |
| 연락처/소개 | ✅ |
| 의료 전문가 프로필 (E-E-A-T) | ✅ |
| 구조화 데이터 | ✅ |
| 의료 출처 표기 | ✅ |
| 자동 광고(Auto ads) 활성화 | ❌ 미설정 |

**액션:**
1. AdSense 대시보드에서 Auto ads 활성화 (승인 확률 향상)
2. kidsfever.xyz 재심사 요청
3. 승인 후 광고 유닛 생성 → 코드에 컴포넌트 추가

#### 1-B. AdSense 승인 후 코드 구현
- `src/app/components/ads/GoogleAdSense.tsx` 반응형 광고 유닛 컴포넌트 생성
- 배치 위치:
  1. 계산 폼~결과 사이 (기존 KakaoAdFit과 교체 또는 병행)
  2. FAQ 섹션 아래
  3. 블로그 글 본문 중간/하단
- `data-ad-format="auto"`, `data-full-width-responsive="true"` 사용

#### 1-C. KakaoAdFit 유지 판단
현재 KakaoAdFit은 정상 작동 중이나 트래픽 부족으로 수익 0원.
- AdSense 승인 전까지 유지
- AdSense 승인 후에는 **AdSense vs KakaoAdFit RPM 비교** 후 결정
- 한국 트래픽 특성상 양쪽 병행도 가능 (광고 슬롯 분리)

### Phase 2: 트래픽 성장 — SEO 콘텐츠 확대 (가장 중요)

> **현재 수익 0원의 근본 원인이 트래픽 부족이므로, 이 단계가 전체 전략의 핵심.**

#### 2-A. 롱테일 키워드 타겟 블로그 콘텐츠

현재 사이트에 블로그 글이 일부 있으나, 검색 유입을 늘리려면 추가 콘텐츠가 필요.

**추천 콘텐츠 주제 (검색량 높고 경쟁 낮은 육아+해열제 키워드):**

| 키워드 | 콘텐츠 유형 | 예상 효과 |
|-------|-----------|---------|
| "체중별 타이레놀 용량표" | 표/인포그래픽 | 계산기로 자연 연결 |
| "부루펜 vs 타이레놀 차이점" | 비교 글 | 정보성 검색 유입 |
| "아기 해열제 교차복용 방법" | 가이드 | 이미 FAQ에 일부 있으나 별도 심화 글 |
| "소아 발열 응급실 기준" | 가이드 | 높은 검색 의도 |
| "아기 해열제 먹이는 법" | 실용 가이드 | 초보 부모 타겟 |
| "해열제 복용 간격 계산" | 도구+설명 | 계산기 기능 연계 |
| "어린이 타이레놀 vs 챔프시럽" | 제품 비교 | 구매 의도 키워드 |

**각 블로그 글 구성:**
- 본문에 계산기 CTA(Call-to-Action) 삽입 → 본 페이지 유입 증가
- 글 중간/하단에 AdSense 광고 슬롯
- 의료 출처 표기 유지 (E-E-A-T)
- FAQ 구조화 데이터 추가 (검색 스니펫 노출)

#### 2-B. 네이버 SEO 강화
현재 네이버 CTR 4.8%는 양호한 편. 개선 여지:
- **메타 디스크립션 최적화**: 검색 결과에서 클릭을 유도하는 문구
- **네이버 블로그 시너지**: 네이버 블로그(blog.naver.com/kimsol1015)에서 kidsfever.xyz로 링크
- **네이버 웹마스터 도구**: 색인 상태 점검, 사이트맵 재제출

#### 2-C. Google 검색 유입 확대
- Google Search Console에서 현재 색인 상태 확인
- 영문 콘텐츠 활용 (이미 i18n 지원) → 해외 트래픽
- AEO (Answer Engine Optimization): AI 검색(Google AI Overview, Bing Chat)에 인용되는 구조화된 답변 작성

## 6. 기술 구현 상세 (코드 변경이 필요한 항목)

### 6-1. Google AdSense 광고 유닛 컴포넌트 (승인 후)

```
신규: src/app/components/ads/GoogleAdSense.tsx

```

### 6-2. 광고 배치 레이아웃
```
[계산 폼]
├── KakaoAdFit 배너 (기존 유지)
[계산 결과]
├── AdSense 반응형 유닛 (신규)
[FAQ 스니펫]
├── AdSense 또는 KakaoAdFit (신규)
[Pieceful 배너] (기존 유지)
```

### 6-3. 블로그 글 광고 배치
```
[블로그 헤더]
[본문 상단]
├── AdSense 인피드 광고 (신규)
[본문 중간]
├── AdSense 인아티클 광고 (신규)
[본문 하단]
[관련 글 추천]
```

### 검증 방법
1. `npm run build` — 빌드 성공 확인
2. `npm run dev` — localhost에서 광고 렌더링 확인
3. 모바일/데스크톱 반응형 테스트
4. 광고 로딩 실패 시 계산기 정상 작동 확인 (의료 안전)
5. CLS 영향 없는지 확인 (고정 높이 예약)
6. `npm test` — 기존 테스트 통과 확인

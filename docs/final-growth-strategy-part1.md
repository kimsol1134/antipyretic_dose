# 최종 마케팅 성장 전략 (개정판)

**버전:** 2.0 (비판적 검토 반영)
**작성일:** 2025-11-15
**대상:** 어린이 해열제 복용량 계산기
**실행 주체:** 1인 의사 창업자 (주 10-15시간 가능)

---

## Executive Summary

### 초안 vs 최종안 핵심 변경사항

| 항목 | 초안 | 최종안 | 변경 이유 |
|-----|------|--------|----------|
| **전략 방향** | B2C 트래픽 중심 | **B2B + B2C 혼합** | 안정적 수익 확보 |
| **Phase 1 목표** | 50,000 방문/월 | **15,000 방문/월** | 현실화 |
| **실행 항목** | 10개 병렬 | **3개 집중** | 리소스 제약 |
| **예산** | $770/월 | **$1,200/월** | 법률·보험 추가 |
| **타임라인** | 12개월 | **24개월** | SEO 현실 반영 |
| **North Star Metric** | 트래픽 | **WAU (주간 활성 사용자)** | 가치 중심 |
| **피봇 기준** | 없음 | **6개월/12개월 체크포인트** | 리스크 관리 |

### 전략 철학 변경

**초안:** "트래픽 → 수익" (Volume Game)
**최종안:** "가치 → 신뢰 → 수익" (Quality Game)

**핵심 원칙:**
1. **Focus over Breadth** - 10개 하는 것보다 3개 잘하기
2. **B2B First** - 안정적 수익 기반 확보 후 B2C 확장
3. **Lean Execution** - 최소 리소스로 PMF 검증
4. **Legal First** - 의료사고 리스크 최우선 대응

## 1. 재정의된 목표 및 KPI

### 1.1 North Star Metric

**주간 활성 계산기 사용자 (Weekly Active Users, WAU)**

정의: 일주일 동안 계산기를 1회 이상 사용한 순수 사용자 수

선정 이유:
- 트래픽보다 **실제 가치 제공** 측정
- 재방문율, 만족도와 직결
- 수익 conversion의 선행 지표

**목표:**
- **Phase 1 (6개월):** WAU 500
- **Phase 2 (12개월):** WAU 2,000
- **Phase 3 (24개월):** WAU 5,000

### 1.2 수정된 Phase별 목표

#### Phase 1: Foundation & Validation (0-6개월)

**주요 목표:**
1. 법적 리스크 제거 (보험 가입, 법률 검토 완료)
2. Product-Market Fit 검증 (WAU 500 달성)
3. B2B 파일럿 고객 2곳 확보
4. 핵심 SEO 기반 구축 (DA 15+, 백링크 30+)

**트래픽 목표:**
- 월 방문자: 5,000 → 15,000 (3배 성장)
- Organic traffic 비중: 60%+

**수익 목표:**
- AdSense + 쿠팡: $50/월
- B2B 구독: $100/월
- **총: $150/월**

#### Phase 2: Growth & Optimization (7-12개월)

**주요 목표:**
1. SEO 상위 진입 (주요 키워드 Top 10)
2. B2B 고객 5곳 확보
3. 프리미엄 기능 MVP 출시
4. 커뮤니티 형성 (이메일 리스트 1,000명)

**트래픽 목표:**
- 월 방문자: 15,000 → 50,000 (3.3배 성장)
- WAU: 500 → 2,000

**수익 목표:**
- Organic 광고 수익: $150/월
- B2B 구독: $250/월
- 프리미엄 구독: $100/월
- **총: $500/월**

#### Phase 3: Scale & Monetization (13-24개월)

**주요 목표:**
1. 육아 건강 플랫폼 확장 (성장 차트, 예방접종)
2. B2B 확대 (10개 기관)
3. 제휴 마케팅 다각화
4. Exit 옵션 준비 (M&A or 투자 유치)

**트래픽 목표:**
- 월 방문자: 50,000 → 150,000 (3배 성장)
- WAU: 2,000 → 5,000

**수익 목표:**
- Organic 광고: $500/월
- B2B: $500/월
- 프리미엄: $300/월
- 제휴: $200/월
- **총: $1,500/월**

## 2. 집중 전략 (Pareto 80/20)

### 2.1 집중할 3가지 핵심 활동

1인 창업자가 **주 10-15시간** 투자로 최대 효과를 내는 활동:

#### 활동 1: 고품질 SEO 콘텐츠 (주 5시간)

**목표:** 구글 검색 순위 상승

**방법:**
- **월 2개 블로그 글** (초안 대비 1/6로 축소)
- 각 2,500-3,000자, 의료 전문성 + SEO 최적화
- 외주 작가 활용 (의료 라이터), 본인은 감수만

**주제 선정 기준:**
- 검색량 500+/월 (Google Keyword Planner)
- 경쟁 난이도 Medium 이하 (Ahrefs KD < 30)
- Featured Snippet 기회 있음

**우선순위 주제 (첫 6개월):**
1. "10kg 아기 타이레놀 용량 - 체중별 완벽 가이드"
2. "타이레놀 vs 부루펜 차이점 (의사 설명)"
3. "해열제 교차복용 방법 - 간격과 주의사항"
4. "밤에 아이 열날 때 체크리스트 (응급실 증상)"
5. "타이레놀 160mg/5mL vs 80mg/mL 농도 차이"
6. "아세트아미노펜 과다복용 증상 및 대처법"

**예산:** 의료 라이터 $400/월 (글 2개 × $200)

#### 활동 2: 백링크 확보 (주 3시간)

**목표:** Domain Authority 상승 (DA 15+)

**방법:**

**2-1. HARO (Help A Reporter Out) - 주 2시간**
- 매일 HARO 이메일 모니터링
- 육아/의료 관련 기자 문의에 답변
- 기사 인용 시 링크 획득 (DA 70+ 매체)

예상 결과:
- 월 15-20개 답변 → 1-2개 기사 인용
- 6개월: 고품질 백링크 6-12개

**2-2. Guest Posting - 월 1회 (시간: 주 1시간)**
- 육아 블로그, 의료 매거진에 기고
- 바이오에 사이트 링크
- 예: 네이버 블로그 "육아의 신", "맘스다이어리"

예상 결과:
- 6개월: 백링크 6개 (DA 20-40)

**총 6개월 예상 백링크: 12-18개 (고품질)**

#### 활동 3: B2B 영업 (주 2시간)

**목표:** 안정적 수익 기반 ($100/월 → $500/월)

**타겟:** 소아과 병원/약국 10곳

**영업 전략:**

**Phase 1 (1-3개월): 무료 파일럿**
- 지인 소아과 의사 3명 연락
- 3개월 무료 트라이얼 제안
- 대기실 태블릿 설치

**Phase 2 (4-6개월): 유료 전환**
- 파일럿 2곳 중 1곳 유료 전환 (성공률 50%)
- 월 $50 구독료

**Phase 3 (7-12개월): 레퍼런스 확대**
- 성공 사례 정리 (A병원 환자 만족도 상승 등)
- 추가 5곳 영업 (레퍼런스 기반)
- 6개월 내 총 5곳 계약 목표

**화이트라벨 패키지:**
- 병원 로고 삽입
- 커스텀 도메인 (clinic.example.com)
- 관리자 대시보드 (사용 통계)

**가격:** 월 $50 (연간 결제 시 $500, 2개월 할인)

### 2.2 삭제할 활동 (비효율적)

초안에서 삭제되는 항목:

❌ **인스타그램** - 시간 대비 ROI 낮음
- 릴스 제작 주 7시간 → 트래픽 전환 <1%
- 대신: B2B 레퍼런스 이미지만 게시 (월 1회, 30분)

❌ **유튜브** - 제작 시간 과다
- 영상 편집 주 8시간 → 삭제
- 대신: 블로그 글을 Text-to-Speech로 자동 변환 (AI 도구)

❌ **네이버 지식iN** - 시간 대비 효과 낮음
- 주 5개 답변 (3시간) → 삭제
- 네이버 검색 유입은 블로그로 대체

❌ **맘카페 활동** - 광고로 인식, 역효과
- 주 3시간 → 삭제
- 대신: 자연스러운 입소문 유도 (사용자가 공유하도록)

❌ **바이럴 캠페인** - 리소스 과다
- 챌린지, PDF 배포 → 삭제 (Phase 2 이후 고려)

**절감 시간: 주 21시간 → 주 10시간으로 축소**

## 3. 법적 리스크 최우선 대응

### 3.1 즉시 실행 필수 (1개월 내)

#### 1. 전문직 배상책임보험 가입

**보험사:** 메리츠, KB손해보험, 현대해상
**보장 내용:**
- 의료 정보 제공으로 인한 손해배상
- 변호사 비용 포함
- 보장 금액: 최소 1억 원

**예상 보험료:** 월 $120-200

#### 2. 법률 자문 ($2,000, 1회)

**검토 항목:**
- 면책 조항 법적 효력 검증
- 이용약관 개정
- 개인정보처리방침 강화
- 의료법 위반 여부 (의료행위 vs 정보 제공 경계)

**변호사:** 의료 전문 법무법인 (서울 소재)

#### 3. 의료 감수 프로세스 구축

**감수자:** 소아과 전문의 2명 (제3자)
**감수 대상:**
- products.json 데이터
- 모든 블로그 콘텐츠
- FAQ 답변

**프로세스:**
1. 콘텐츠 작성 → 2. 감수자 검토 → 3. 수정 → 4. 최종 승인 → 5. 발행

**비용:** 건당 $50 (월 2개 × $50 = $100/월)

#### 4. 사용자 동의 절차 추가

**변경사항:**
- 계산기 사용 전 면책 동의 체크박스 추가
- "본 계산기는 참고용이며, 실제 투약 전 의사와 상담하세요" 팝업
- 동의 로그 저장 (법적 증거)

**기술 구현:** 2주 (본인 개발)

### 3.2 데이터 신뢰성 확보

#### 분기별 데이터 검증 절차

**책임자:** 본인 (solkim)
**주기:** 3개월마다

**검증 항목:**
1. FDA 가이드라인 변경 확인
2. 식약처 허가사항 업데이트 확인
3. 제품 단종/신제품 출시 확인
4. 농도 변경 여부 확인

**알림 시스템:**
- FDA RSS 구독
- 식약처 이메일 알림 신청
- Google Alerts 설정 ("타이레놀 용량 변경")

#### 버전 관리 (products.json)

**변경 이력:**
```json
{
  "version": "2.1",
  "lastUpdated": "2025-03-15",
  "changelog": [
    {
      "date": "2025-03-15",
      "changes": "FDA guideline updated: max single dose 650mg → 600mg"
    }
  ]
}
```

**사용자 공지:**
- 사이트 하단: "최종 업데이트: 2025-03-15"
- 계산 결과 하단: "이 정보는 2025-03-15 기준입니다"

## 5. 피봇 기준 (Kill Criteria)

### 5.1 6개월 체크포인트

**측정 지표:**
- WAU < 200
- 월 방문자 < 5,000
- B2B 계약 0개
- 수익 < $50/월

**피봇 옵션 1: B2B 전환**
- B2C 마케팅 중단
- B2B 영업에 100% 집중
- 목표: 6개월 내 10개 병원 계약 ($500/월)

**피봇 옵션 2: 유료 광고 테스트**
- Google Ads $300/월 투자
- 3개월 테스트
- ROI < 1.5 → 중단

**피봇 옵션 3: 프로젝트 중단**
- 조건: 12개월 누적 수익 < $1,000
- 사이트 유지 (최소 관리 모드)
- 매각 시도 (가격: $2,000-5,000)

### 5.2 12개월 체크포인트

**측정 지표:**
- WAU < 1,000
- 월 방문자 < 20,000
- B2B 계약 < 3개
- 수익 < $300/월

**피봇 옵션 1: 매각**
- 잠재 바이어: 육아 플랫폼, 병원 체인, 제약회사
- 가격: 연 수익 × 2-3배 ($5,000-10,000)
- 플랫폼: Flippa, Empire Flippers

**피봇 옵션 2: 제품 확장**
- 성장 차트 계산기 추가
- 예방접종 스케줄러 추가
- 육아 건강 플랫폼으로 리브랜딩

**피봇 옵션 3: 유지 (Lifestyle Business)**
- 최소 관리 (월 5시간)
- 수익 $200-500/월 유지
- 사이드 프로젝트로 전환

## 6. 경쟁 우위 재구축

### 6.1 지속 가능한 차별화 요소

초안의 "의사가 만든 도구"만으로는 부족. 추가 차별화:

#### 1. 의료 커뮤니티 승인

**목표:** 대한소아과학회 추천 획득

**방법:**
- 학회에 도구 소개 (이메일, 발표)
- 무료 제공 (비영리 명시)
- 추천 받으면 사이트에 로고 게시

**효과:**
- 신뢰도 폭발적 상승
- 의사들이 환자에게 추천
- 언론 보도 기회

#### 2. 오픈소스화

**전략:** GitHub에 소스코드 공개

**이유:**
- 투명성 → 신뢰도 상승
- 개발자 커뮤니티 기여 (버그 수정, 기능 추가)
- 기술 블로그 백링크 (Hacker News, Reddit)

**라이선스:** MIT (상업적 이용 허용, 책임 면제)

#### 3. 데이터 API 제공

**전략:** 계산 API 무료 공개

**용도:**
- 다른 육아 앱이 통합 가능

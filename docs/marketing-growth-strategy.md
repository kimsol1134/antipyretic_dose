## Executive Summary

### 핵심 발견사항

**강점:**
- 의사가 만든 의료 도구로 높은 신뢰도 (E-A-T 최적화)
- IEC 62304 의료기기 소프트웨어 안전 기준 준수
- 완벽한 SEO 최적화 (메타태그, 구조화 데이터, Featured Snippet 타겟팅)
- 양질의 기술 스택 (Next.js 15, React 19, TypeScript)
- 이중 언어 지원 (한국어/영어)으로 시장 확장 가능
- 이미 수익화 구조 갖춤 (AdSense + 쿠팡 파트너스)

**현재 한계:**
- 트래픽 데이터 부재 (실제 유입량 미확인)
- 사용자 참여 메커니즘 부족 (이메일 수집, 계정 없음)
- 소셜 미디어 존재감 전무
- 백링크 전략 부재
- 브랜드 인지도 낮음
- 단일 기능 도구 (계산기만 제공)

**시장 기회:**
- 한국 연간 출생아 수 약 24만 명 (2024년 기준)
- 미국 연간 출생아 수 약 360만 명
- 육아 정보 검색 시장 지속 성장
- 네이버/구글 검색 트래픽 선점 가능
- 의료 정보 신뢰성 문제로 인한 공신력 있는 도구 수요 증가

## 1. 현재 상태 심층 분석

### 1.1 제품 및 기술 현황

**핵심 가치 제안:**
- 체중·나이별 정확한 해열제 복용량 즉시 계산
- 7개 주요 제품 지원 (타이레놀, 챔프, 부루펜, 맥시부펜, Advil, Motrin 등)
- 식약처/FDA 기준 기반 안전성 검증
- 무료, 로그인 불필요, 즉시 사용 가능

**기술적 우수성:**
- Server Components로 빠른 초기 로딩
- Build-time 데이터 검증으로 잘못된 정보 배포 방지
- Core Web Vitals 최적화 (CLS, INP, LCP)
- 모바일 최적화 (decimal keyboard, 반응형 UI)

**현재 콘텐츠 자산:**
- 메인 계산기 페이지 (한/영)
- FAQ 8+ 문항 (의료적 근거 포함)
- About, Terms, Privacy, Contact 페이지
- 유사 약품 정보 (한국판)
- 관련 제품 정보 (미국판)

### 1.2 SEO 및 검색 가시성 분석

**SEO 우수 사항:**
✅ 완벽한 메타데이터 (Title, Description, Keywords)
✅ Open Graph + Twitter Card 설정
✅ JSON-LD 구조화 데이터 (MedicalWebPage + WebApplication)
✅ Google + Naver 검색엔진 최적화
✅ Canonical URL + Hreflang 설정
✅ FAQ Featured Snippet 타겟팅 (40-60자 짧은 답변)
✅ 의료적 출처 명시 (Seoul Asan Hospital, FDA, AAP)

**SEO 개선 필요 사항:**
❌ 백링크 거의 없음 (Domain Authority 낮음 예상)
❌ 블로그 콘텐츠 없음 (롱테일 키워드 미확보)
❌ 사용자 리뷰/평가 시스템 없음
❌ 소셜 공유 버튼 없음
❌ 페이지 로딩 속도 측정 필요 (Lighthouse 점수 미확인)

**검색 키워드 기회 분석:**

| 키워드 타입 | 예시 | 검색 의도 | 우선순위 |
|------------|------|----------|---------|
| **Head Keywords** | 타이레놀 복용량, 어린이 해열제 | 정보 탐색 | 높음 |
| **Long-tail Keywords** | 10kg 아기 타이레놀 용량, 18개월 부루펜 몇ml | 즉시 답변 필요 | 매우 높음 |
| **Comparison Keywords** | 타이레놀 vs 부루펜, 아세트아미노펜 이부프로펜 차이 | 비교 정보 | 중간 |
| **Safety Keywords** | 해열제 교차복용, 타이레놀 최대 용량, 부루펜 부작용 | 안전성 확인 | 높음 |
| **Problem-solving** | 아이 열 안떨어져요, 해열제 안듣어요 | 해결책 탐색 | 중간 |

### 1.3 사용자 경험 및 전환 퍼널 분석

**현재 사용자 여정:**
```
검색엔진 랜딩 → 계산기 사용 → 결과 확인 → 이탈
```

**주요 문제점:**
1. **One-time Use 패턴**: 재방문 유도 메커니즘 없음
2. **No Engagement Loop**: 이메일 수집, 저장 기능, 히스토리 없음
3. **No Sharing Incentive**: 공유 버튼, 소셜 증거 부재
4. **No Community**: 커뮤니티, 댓글, Q&A 없음
5. **Limited Monetization**: 쿠팡 배너 1개 + AdSense만 의존

**전환 지표 (추정):**
- 방문자 → 계산기 사용: 60-70% (예상)
- 계산기 사용 → 재방문: 10% 미만 (예상)
- 방문자 → 광고 클릭: 1-3% (업계 평균)
- 방문자 → 쿠팡 구매: 0.5-1% (예상)

## 2. 타겟 시장 및 페르소나 분석

### 2.1 Primary Target: 초보 부모 (0-3세 자녀)

**인구통계:**
- 연령: 28-40세
- 성별: 엄마 70%, 아빠 30%
- 소득: 중산층 이상
- 거주지: 도시 및 수도권 집중

**행동 특성:**
- 야간/새벽 긴급 검색 빈번 (아이 열날 때)
- 모바일 검색 80% 이상
- 네이버 카페, 맘카페 활발히 이용
- 인스타그램, 유튜브에서 육아 정보 소비
- 의사/약사 의견 높이 신뢰

**Pain Points:**
- 밤에 아이 열나면 당황, 용량 계산 어려움
- 약통 뒷면 용량표 읽기 복잡
- 인터넷 정보 신뢰도 낮음
- 병원 갈 정도는 아닌데 불안함

**검색 키워드 예시:**
- "10kg 아기 타이레놀 용량"
- "18개월 부루펜 몇ml"
- "타이레놀 4시간마다 괜찮아요?"
- "해열제 교차복용 방법"

### 2.2 Secondary Target: 경력 부모 (2명 이상 자녀)

**특징:**
- 경험 있지만 정확한 용량 재확인 필요
- 빠른 계산 도구 선호
- 북마크/재방문 확률 높음
- 지인 추천 의향 높음

### 2.3 Tertiary Target: 의료/보육 종사자

**특징:**
- 소아과 간호사, 약사, 보육교사
- 보호자 상담 시 정확한 정보 제공 필요
- B2B 잠재 고객 (병원/약국 링크 게시)
- 전문가 추천으로 신뢰도 상승

## 3. 성장 전략 로드맵

#### 3.1.1 콘텐츠 SEO 강화

**블로그 콘텐츠 제작 (우선순위 높음):**

1. **롱테일 키워드 타겟 포스트 (20개 작성)**
   - "체중별 타이레놀 복용량표 (5kg~30kg 완벽 가이드)"
   - "생후 4개월 아기 열날 때 대처법 (타이레놀 용량 포함)"
   - "타이레놀 vs 부루펜 차이점 총정리 (의사 설명)"
   - "해열제 교차복용 완벽 가이드 (간격, 순서, 주의사항)"
   - "밤에 아이 열날 때 체크리스트 (응급실 가야 할 증상)"
   - "타이레놀 160mg/5ml vs 80mg/0.8ml 농도 차이 설명"
   - "소아 이부프로펜 부작용 및 주의사항"
   - "아세트아미노펜 과다복용 증상 및 대처법"
   - "해열제 먹여도 열 안떨어질 때 (원인 5가지)"
   - "신생아 열 vs 영아 열 (월령별 대처법)"

2. **FAQ 확장 (현재 8개 → 30개 목표)**
   - 구글 "People Also Ask" 분석
   - 네이버 지식iN 인기 질문 수집
   - 맘카페 자주 묻는 질문 정리

3. **비교 콘텐츠 (Comparison Pages)**
   - 타이레놀 vs 챔프 vs 부루펜 3-way 비교표
   - 국내 vs 해외 제품 비교
   - 브랜드별 가격 비교 (쿠팡 링크 연결)

**기술적 SEO 개선:**
- [ ] Lighthouse 100점 목표 (Performance, Accessibility, SEO)
- [ ] Schema.org FAQ 마크업 추가
- [ ] Breadcrumb navigation 구조화 데이터
- [ ] Image alt text 최적화 (제품 이미지)
- [ ] robots.txt 세분화 (크롤링 최적화)
- [ ] Sitemap.xml 동적 생성 (블로그 포함)

#### 3.1.2 백링크 전략

**Tier 1: 고품질 백링크 (DA 50+)**
- 네이버 블로그 (의사/약사 전문가 블로그)
- 의료 포털 (질병관리청, 서울아산병원 블로그)
- 육아 커뮤니티 (맘스홀릭, 베이비트리)
- 언론사 기고 (헬스조선, 메디칼타임즈)

**실행 계획:**
1. **HARO (Help A Reporter Out) 활용**
   - 기자 문의에 전문가 답변 제공
   - 기사 내 사이트 링크 획득

2. **의료 전문가 네트워킹**
   - 소아과 의사 100명에게 이메일 아웃리치
   - "선생님 블로그에 무료 도구 소개해주세요" 제안
   - 병원 홈페이지 "유용한 링크" 섹션 등록

3. **육아 인플루언서 협업**
   - 인스타그램 육아 인플루언서 20명 선정
   - 무료 도구 소개 + 스토리 공유 요청
   - 링크 in bio 또는 블로그 링크

4. **디렉토리 등록**
   - 네이버 의료정보 디렉토리
   - Google My Business (해당 시)
   - 육아 리소스 디렉토리 10곳

**Tier 2: 커뮤니티 참여**
- 맘카페 (네이버 카페 Top 20)
- 육아 커뮤니티 (아이캔디, 맘스홀릭)
- Reddit r/Parenting, r/beyondthebump
- Facebook 육아 그룹

**실행 방법:**
- 직접 광고 금지, 자연스러운 멘션
- "이 계산기 써보니 정확하더라고요" 댓글 유도
- Q&A에 답변하며 링크 제공

#### 3.1.3 네이버 최적화 (한국 시장)

**네이버 검색 점유율 60%+ → 필수 최적화**

4. **네이버 카페 진출**

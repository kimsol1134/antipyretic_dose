## 9. SEO 콘텐츠 제작 가이드

### 9.1 월 2개 블로그 글 제작 프로세스

**타임라인 (월):**
```
Week 1: 주제 선정 + 키워드 리서치 (2시간)
Week 2: 외주 작가에게 초안 의뢰 (작가 작업 5일)
Week 3: 의료 감수 + 수정 (3시간)
Week 4: 발행 + 네이버 블로그 재가공 (2시간)
```

#### Step 1: 주제 선정 (월초, 2시간)

**도구:**
- Google Keyword Planner (무료)
- Ahrefs Keywords Explorer ($100/월)
- 네이버 키워드 도구 (무료)

**선정 기준:**
1. **검색량:** 월 300+ (너무 많으면 경쟁 치열)
2. **경쟁 난이도:** KD < 30 (Ahrefs 기준)
3. **Featured Snippet 기회:** "People Also Ask" 박스 있음
4. **상업적 가치:** 제품 구매로 이어질 수 있는 키워드

**예시:**

| 키워드 | 검색량 | KD | Featured Snippet | 우선순위 |
|-------|-------|----|--------------------|---------|
| "10kg 타이레놀 용량" | 800/월 | 25 | ✅ | 높음 |
| "해열제 교차복용" | 600/월 | 30 | ✅ | 높음 |
| "타이레놀 부작용" | 1,200/월 | 45 | ❌ | 중간 |
| "아기 열" | 5,000/월 | 60 | ❌ | 낮음 |

#### Step 2: 콘텐츠 브리프 작성 (1시간)

**외주 작가에게 전달할 브리프:**

```markdown
## 콘텐츠 브리프

**제목:** 10kg 아기 타이레놀 용량 - 체중별 완벽 가이드

**타겟 키워드:**
- 주요: "10kg 타이레놀 용량"
- 부가: "아기 타이레놀 몇ml", "타이레놀 체중별 용량"

**분량:** 2,500-3,000자

**구조:**
1. 도입 (200자)
   - 밤에 아이 열날 때 부모의 당황
   - 정확한 용량의 중요성
2. Featured Snippet 타겟 답변 (50자)
   - "10kg 아기는 타이레놀 160mg/5mL 기준 5mL 투약"
3. 체중별 용량표 (표 형식)
   - 5kg, 10kg, 15kg, 20kg, 25kg, 30kg
4. 농도별 차이 설명
   - 160mg/5mL vs 80mg/mL
5. 주의사항
   - 최대 용량, 복용 간격
6. FAQ (3-5개)
   - "4시간마다 괜찮나요?"
   - "밤에 깨워서 먹여야 하나요?"
7. CTA
   - 계산기 사용 유도 (링크)

**톤앤매너:**
- 전문적이면서 따뜻함
- 의학 용어는 쉽게 풀어쓰기
- 부모의 불안을 공감

**출처 필수:**
- FDA 가이드라인
- 식약처 허가사항
- 대한소아과학회 자료

**금지 사항:**
- 절대적 표현 ("무조건", "100% 안전")
- 의사 상담 대체하는 표현
- 특정 제품 과도한 홍보
```

#### Step 3: 의료 감수 (Week 3, 2시간)

**감수자:** 소아과 전문의 2명 (교차 검증)

**체크리스트:**
- [ ] 용량 정보 정확성
- [ ] 최신 가이드라인 반영
- [ ] 의학적 오류 없음
- [ ] 면책 조항 포함
- [ ] 출처 명시

**수정 사항 반영:** 작가에게 재작업 의뢰

#### Step 4: SEO 최적화 (발행 전, 1시간)

**On-page SEO 체크리스트:**
- [ ] 제목에 키워드 포함 (앞쪽 배치)
- [ ] Meta Description 160자 이내, 키워드 포함
- [ ] H1, H2, H3 계층 구조
- [ ] 키워드 밀도 1-2% (자연스럽게)
- [ ] 이미지 alt text (키워드 포함)
- [ ] 내부 링크 3개 (계산기, FAQ, 관련 글)
- [ ] 외부 링크 2개 (출처: FDA, 대한소아과학회)
- [ ] Schema.org Article 마크업
- [ ] FAQ Schema (Featured Snippet 타겟)

**기술 SEO:**
- [ ] Canonical URL 설정
- [ ] Open Graph 태그
- [ ] Twitter Card
- [ ] Loading speed < 2초 (GTmetrix 확인)

#### Step 5: 발행 및 배포 (Week 4, 1시간)

**배포 채널:**
1. **본 사이트 블로그** (신규 섹션 /blog)
2. **네이버 블로그** (동일 내용, 네이버 형식 재가공)
3. **이메일 뉴스레터** (구독자에게 발송)

**소셜 미디어 (최소화):**
- 페이스북: 자동 포스팅 (Buffer 사용)
- 트위터: 링크만 공유

## 10. 백링크 확보 실행 매뉴얼

### 10.1 HARO (Help A Reporter Out) 활용법

**Step 1: HARO 가입 (무료)**
- 웹사이트: helpareporter.com
- 이메일 등록
- 카테고리 선택: Healthcare, Lifestyle (육아 관련)

**Step 2: 일일 루틴 (15분/일)**
- 아침 9시: HARO 이메일 확인
- 육아/의료 관련 기자 문의 찾기
- 적합한 질문에 답변 작성

**답변 작성 템플릿:**
```
Subject: RE: [Query Title] - Pediatrician Response

Hi [Reporter Name],

I'm Dr. [Your Name], a pediatrician and creator of [Your Site],
a dosage calculator trusted by [X] parents monthly.

Regarding your question about [Topic]:

[50-100 words expert answer]

[Supporting data/study if available]

I'm happy to provide further insights or be quoted in your article.

Best regards,
Dr. [Your Name]
[Your Site URL]
[LinkedIn Profile]
```

**성공률 높이는 팁:**
- 빠른 응답 (문의 후 1시간 이내)
- 구체적 데이터 제공 (연구 결과, 통계)
- 짧고 명확한 답변 (100-150자)
- 의사 자격 강조

**예상 결과:**
- 월 20개 답변 → 1-2개 기사 인용
- 백링크 DA 70+ (Forbes, HuffPost 등)

## 11. B2B 영업 플레이북

### 11.1 타겟 고객 리스트 작성

**우선순위 1: 지인 소아과 의사 (Warm Lead)**
- 의대 동기, 선후배
- 병원 근처 약국

**우선순위 2: 중소형 소아과 (Cold Lead)**
- 서울/수도권 개인 소아과 100곳
- 직원 5명 이하 (의사 결정 빠름)

**우선순위 3: 약국 체인**
- 온누리약국, 메디팜, 씨엔팜

**리스트 수집 방법:**
- 네이버 지도 검색 "소아과"
- 건강보험심사평가원 의료기관 검색
- 대한소아과학회 회원 명단

**Excel 시트 관리:**
```
| 병원명 | 원장명 | 전화번호 | 이메일 | 접촉일 | 상태 | 비고 |
```

### 11.2 영업 프로세스 (3단계)

#### Phase 1: 초기 접촉 (이메일 + 전화)

**이메일 템플릿:**
```
Subject: 대기실 환자 서비스 개선 제안 - 무료 트라이얼

안녕하세요, [원장님 성함]님

저는 소아과 의사 pinecone이라고 합니다.

선생님 병원 환자분들을 위한 무료 도구를 소개하고자 합니다.

[사이트명]은 어린이 해열제 용량을 체중별로 자동 계산해주는
웹 도구로, 월 [X]명의 부모님들이 사용하고 있습니다.

대기실 태블릿에 설치하시면:
- 환자 대기 시간 활용 (교육 콘텐츠 제공)
- 보호자 만족도 상승
- 진료 시 용량 문의 감소

3개월 무료 트라이얼 제안드립니다.
관심 있으시면 회신 부탁드립니다.

Dr. pinecone
[연락처]
```

**전화 후속 (이메일 발송 3일 후):**
```
"안녕하세요, 지난주에 이메일 드린 pinecone입니다.
대기실 환자 서비스 도구 관련해서 간단히 설명드리고 싶어서요.
2-3분만 시간 괜찮으실까요?"
```

#### Phase 2: 데모 & 설치 (30분 방문 or 화상)

**준비물:**
- 태블릿 (직접 시연)
- 1페이지 설명 자료 (PDF)
- 사용 통계 (타 병원 사례)

**스크립트:**
```
"보호자들이 가장 많이 묻는 질문이 뭔가요?
대부분 '우리 아이 몇 ml 먹이나요?'죠.

이 도구를 대기실에 두시면, 보호자가 스스로 확인하고
진료실에 들어와서 '5ml가 맞나요?'만 확인하면 됩니다.

선생님 시간 절약 + 보호자 만족도 상승 효과가 있습니다.

3개월 무료로 써보시고, 만족하시면 월 5만원에 계속 쓰시면 됩니다."
```

#### Phase 3: 유료 전환 (3개월 후)

**유료 전환 이메일:**
```
Subject: [병원명] 무료 트라이얼 종료 안내

안녕하세요, [원장님]님

3개월 무료 트라이얼 기간이 종료됩니다.

사용 통계:
- 총 사용 횟수: 128회
- 일평균 사용: 1.4회
- 보호자 만족도: (설문 결과)

계속 사용하시려면 월 5만원 (연간 결제 시 50만원, 2개월 할인)입니다.

결제 링크: [Stripe 링크]

감사합니다.
```

**예상 전환율: 50%** (3곳 중 1.5곳)

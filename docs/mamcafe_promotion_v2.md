### 💡 해결책 찾았어요!

**체중·나이만 입력하면 끝나는 해열제 계산기**

#### 이것만 입력하면:
- 아이 체중 (kg)
- 나이 (개월 또는 만 나이)

#### 이것을 바로 알려줘요:
✅ 타이레놀 정확한 ml
✅ 부루펜 정확한 ml
✅ 챔프/맥시부펜 정확한 ml
✅ 최대 1회 복용량 체크
✅ 연령별 복용 가능 여부

### 📊 신뢰할 수 있는 이유

1. **식약처 공식 데이터** 기반 (e약은요 API 연동)
2. **체중 기준** 그대로 적용 (체중 kg당 용량)
3. **각 제품별 농도** 자동 반영
4. **안전 장치** 내장 (최대 용량 초과 방지)

### 🚀 지금 바로 사용해보세요

**[링크] 우리아이 해열제 계산기**

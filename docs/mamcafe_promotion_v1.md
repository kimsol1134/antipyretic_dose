# 맘카페 홍보글 버전 1: 공감형 스토리텔링

## 제목: 새벽 3시, 아이 열나서 당황했던 의사 남편이 만든 해열제 계산기

안녕하세요, 두 돌 아기 엄마예요.

**"38.5도... 타이레놀 얼마나 먹여야 하지?"**

지난 새벽, 갑자기 열이 펄펄 끓는 아이를 보며 손이 떨렸어요.
약통 뒤에 적힌 설명서는 너무 작아서 안 보이고, 인터넷 검색하니 다 다른 정보들...
결국 응급실 갔다가 "체중 대비 용량 계산하면 돼요"라는 의사 선생님 말씀만 듣고 돌아왔죠.

그날 이후로 저는 결심했어요.
**"다른 엄마들은 나처럼 당황하지 않았으면 좋겠다"**

### 🌡️ 그래서 만들었습니다

**우리 아이 체중과 나이만 입력하면 바로 계산되는 해열제 복용량 계산기!**
- 타이레놀 (아세트아미노펜)
- 부루펜 (이부프로펜)
- 챔프/맥시부펜 (덱시부프로펜)

식약처 공식 데이터 기반으로 정확하게 계산해드려요.

### 💝 이런 분들께 꼭 필요해요

- "아이 체중 12.5kg인데... 타이레놀 몇 ml 먹이지?" 매번 헷갈리시는 분
- "부루펜이랑 타이레놀 교대로 먹이는데 용량이 달라서 헷갈려요" 하시는 분
- "새벽에 열나면 정신없어서 계산도 못하겠어요" 하시는 분
- "약국에서 준 시럽이랑 집에 있는 거랑 농도가 달라요" 당황하신 분

### ✅ 사용해보신 맘들 반응

"북마크 해놨어요! 진짜 유용해요"
"새벽에 아이 열날 때 이거 보고 침착하게 대처했어요"
"남편한테도 공유했어요. 이제 혼자서도 잘 해요"

### 📱 사용 방법 (정말 간단해요!)

1. 아이 체중 입력 (예: 10.5kg)
2. 나이 입력 (개월수 또는 만 나이)
3. 계산 버튼 클릭
→ 타이레놀, 부루펜, 챔프, 맥시부펜 용량이 한번에 나와요!

**🔗 바로가기: [아이 해열제 계산기 링크]**

광고 아니에요. 제가 필요해서 만든 거라 무료예요.
다른 맘들도 도움 되셨으면 좋겠어서 공유합니다.

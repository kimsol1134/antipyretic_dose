# 쿠팡 배너 구현 계획서

## 프로젝트 개요
어린이 해열제 복용량 계산기 웹사이트에 쿠팡 파트너스 배너 광고를 추가하여 수익화

## 문제 정의
이전 구현에서 배너가 왼쪽 아래 구석에만 표시되는 문제 발생
- 원인: 쿠팡 스크립트가 생성하는 iframe이 적절한 컨테이너 없이 삽입됨
- 해결: 중앙 정렬 컨테이너와 반응형 크기 제어 필요

## 배너 배치 위치

### 위치 1: 계산 폼 아래
```
[계산 폼]
↓
[쿠팡 배너]
↓
출처: 식품의약품안전처_의약품개요정보(e약은요) (2025-10-27 검토)
↓
[계산 결과]
```

### 위치 2: 계산 결과 사이 (2개 약마다)
```
[약품 1 결과 카드]
[약품 2 결과 카드]
↓
[쿠팡 배너]
↓
[약품 3 결과 카드]
[약품 4 결과 카드]
```

## 기술 요구사항

### 배너 코드
```html
<script src="https://ads-partners.coupang.com/g.js"></script>
<script>
    new PartnersCoupang.G({
        "id": 936910,
        "trackingCode": "AF3034941",
        "subId": null,
        "template": "carousel",
        "width": "320",
        "height": "100"
    });
</script>
```

### 반응형 크기
- **모바일** (< 768px): 320 x 100
- **데스크톱** (>= 768px): 728 x 90

### 면책 문구
배너 아래에 작은 글씨로 표시:
> 이 사이트는 쿠팡 파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다.

## 주의사항

### 의료 안전성
- 배너가 계산 로직이나 결과 표시에 영향을 주지 않아야 함
- 배너 로딩 실패 시에도 계산기 기능은 정상 작동해야 함

### 성능
- 쿠팡 스크립트는 async 로딩
- 배너가 CLS(Cumulative Layout Shift)를 유발하지 않도록 고정 높이 설정

### 법적 준수
- 면책 문구 필수 표시
- 쿠팡 파트너스 정책 준수
